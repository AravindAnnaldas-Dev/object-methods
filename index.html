Object Creation Methods

Object.create()

To create a new object

Object.assign()

Copying the other source objects and it modify the target object.

Object.fromEntries()

which transforms a list of key-value pairs and return's in to an onject.

Object Property Methods

Object.defineProperty()

To define or modify the property.

Object.defineProperties()

To define ore modify multiple properties.

Object.getOwnPropertyDescriptor()

Which returns the descriptor of an property in an element.

Object.getOwnPropertyDescriptors()

Which returns the multiple decriptors of an multiple properties in an element.

Object.hasOwn()

Returns true/false whether the object has specified property.

Object.keys()

Which returns array of keys(enumerables only) of an object.

Object.values()

Which returns array of values of an object.

Object.entries()

which returns array of key-value pair's of an object.

Object Inspection Methods

Object.is()

which compares two values or objects and returns true/false.

Object.isExtensible()

Returns true/false whether object can accept more properties or not.

Object.isFrozen()

Returns true/ false whether the object has been frozen or not.

Object.isSealed()

Returns true/ false whether the object has been sealed or not.

Object.getPrototypeOf()

Returns proptotype of an object.

Object.setPrototypeOf()

To set the prototype for an object.

Object.getOwnPropertyNames()

Which returns array of keys(enumerables and non-enumerables) of an object.

Object.getOwnPropertySymbols()

Which returns array of symbols of an object.

Object Manipulation Methods

Object.freeze()

Which we cannot add property, cannot delete property and modify property.(No deep freeze)

Object.seal()

Which we cannot add property, cannot delete property but we can modify property.(No deep seal)

Object.preventExtensions()

Which we cannot add properties to an object.

Object Serialization Methods

JSON.stringify(obj)

Coverts object in to an JSON string.

JSON.parse(string)

Coverts JSON string in to an object.

Prototype Methods (Available via Object.prototype)

obj.hasOwnProperty()

Returns true/false whether it has specified property or not.

obj.toString()

Returns string representing an object.

obj.valueOf()

Returns primitive value representing an object.(returns same object)

obj.isPrototypeOf()

whether obj is in the prototype chain of otherObj.

obj.propertyIsEnumerable()

Checks if the specified property is enumerable or not.
